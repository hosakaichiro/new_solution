ディレクトリ移動
cd C:/

フォルダ作成
mkdir git

ディレクトリ移動（gitフォルダに）
cd git

ローカルにgithubのリポジトリクローンを作成する
このときはマスターがすでにgithubにある状態
git clone　git@github.com:hosakaichiro/new_solution.git
↑sshのほう

cdでクローンのリポジトリに移動する

git branchでブランチを確認
最初はmasterのみだと思う

ブランチを切る（マスターから作業ブランチを作成すること）

新しいブランチの作成
git branch 新しいブランチの名前
例えば[work]とかにする

ブランチの切り替え
git checkout ブランチの名前
masterから切り替える
